# 🎬 Netflix Mobile App (Vite + React)

A **mobile-first Netflix UI clone** built with **Vite** and **React**, designed for fast performance and a smooth user experience.  
This project recreates the Netflix mobile interface with dynamic movie sections, responsive layout, and reusable React components.

---

## 🚀 Features

- 📱 **Mobile-first responsive design** – optimized for small screens  
- 🎥 **Hero banner section** with featured movies  
- 📂 **Category-based movie rows** with horizontal scroll  
- 🔍 **Search UI** (static or dynamic depending on your setup)  
- ⚛️ **Reusable React components** for clean code architecture  
- ⚡ **Built with Vite** for fast development and optimized builds

---

## 🛠 Tech Stack

- **React** – for building UI components  
- **Vite** – lightning-fast build tool  
- **JavaScript (ES6+)** – modern JavaScript syntax  
- **CSS / Tailwind / Styled Components** – styling (replace with your setup)  
- Optional: **TMDB API** for dynamic movie data
